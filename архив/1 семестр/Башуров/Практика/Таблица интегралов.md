## 1. Степенные функции

$$
\int x^n\,dx = \frac{x^{n+1}}{n+1} + C,\quad n\neq -1
$$

$$
\int x^{-1}\,dx = \ln|x| + C
$$


## 2. Показательные функции

$$
\int a^x\,dx = \frac{a^x}{\ln a} + C
$$

$$
\int e^x\,dx = e^x + C
$$


## 3. Логарифмы

$$
\int \ln x\,dx = x\ln x - x + C
$$

$$
\int \log_a x\,dx = x\log_a x - \frac{x}{\ln a} + C
$$


## 4. Тригонометрические функции

$$
\int \sin x\,dx = -\cos x + C
$$

$$
\int \cos x\,dx = \sin x + C
$$

$$
\int \tan x\,dx = -\ln|\cos x| + C
$$

$$
\int \cot x\,dx = \ln|\sin x| + C
$$

$$
\int \sec x\,dx = \ln|\sec x + \tan x| + C
$$

$$
\int \csc x\,dx = \ln|\csc x + \cot x| + C
$$

$$
\int \sin^2 x\,dx = \frac{x}{2} - \frac{\sin(2x)}{4} + C
$$

$$
\int \cos^2 x\,dx = \frac{x}{2} + \frac{\sin(2x)}{4} + C
$$


## 5. Обратные тригонометрические функции

$$
\int \frac{dx}{\sqrt{1-x^2}} = \arcsin x + C
$$

$$
\int \frac{dx}{1+x^2} = \arctan x + C
$$

$$
\int \frac{dx}{|x|\sqrt{x^2-1}} = \operatorname{arcsec} x + C
$$


## 6. Гиперболические функции

$$
\int \sinh x\,dx = \cosh x + C
$$

$$
\int \cosh x\,dx = \sinh x + C
$$

$$
\int \tanh x\,dx = \ln|\cosh x| + C
$$


## 7. Рациональные функции

$$
\int \frac{dx}{a^2 + x^2} = \frac{1}{a}\arctan\frac{x}{a} + C
$$

$$
\int \frac{dx}{x^2 - a^2} =
\frac{1}{2a}\ln\left|\frac{x-a}{x+a}\right| + C
$$

$$
\int \frac{dx}{x-a} = \ln|x-a| + C
$$

$$
\int \frac{dx}{(x-a)^n} = -\frac{1}{(n-1)(x-a)^{n-1}} + C
$$


## 8. Корни

$$
\int \sqrt{x}\,dx = \frac{2}{3}x^{3/2} + C
$$

$$
\int \frac{dx}{\sqrt{x}} = 2\sqrt{x} + C
$$

$$
\int x^{m/n}\,dx = \frac{x^{m/n+1}}{m/n+1} + C
$$


## 9. Комбинированные функции

$$
\int e^{ax}\sin bx\,dx =
\frac{e^{ax}}{a^2+b^2}(a\sin bx - b\cos bx) + C
$$

$$
\int e^{ax}\cos bx\,dx =
\frac{e^{ax}}{a^2+b^2}(a\cos bx + b\sin bx) + C
$$


## 10. Часто используемые частные случаи

$$
\int x e^x dx = (x-1)e^x + C
$$

$$
\int x\sin x dx = \sin x - x\cos x + C
$$

$$
\int x\cos x dx = x\sin x + \cos x + C
$$
