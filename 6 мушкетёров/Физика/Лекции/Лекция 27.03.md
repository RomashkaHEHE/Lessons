## Термодинамика
#### Степени свободы газовых молекул
Степень свободы ( $i$ )
    число независимых направлений по которым может двигаться частица в пространстве
Одноатомный газ ($He, Ar$ и другие) 
	имеет один атом в молекуле. Для него $i=3$.
Двухатомный газ ($O_{2}$, $H_{2}$, $N_{2}$, $CO$, $\dots$)
	![[двухатомный]]
	$i = 5=3+2$
Трёхатомный и более ($CO_{2}$, $CH_{4}$, $\dots$)
	![[Трёхатомный]]
В общем случае
	$<W_{кин}> = \frac{i}{2}kT$
Для идеального газа
	$<W_{кин}> = \frac{3}{2}kT$

В термодинамике изучаем:
- Внутреннюю энергию идеального газа ( $u$ ) - функция состояния
- Энтропия ( $S$ ) - функция состояния
- Работа газа ( $A$ ) - функция процесса
- Количество теплоты ( $Q$ ) - функция процесса


#### Работа газа
— при расширении.
Дано:
- идеальный газ
- $p$
- $V$
- $dV$
- $A - ?$
![[задача про газ]]
Т.к. сила давления $\neq const$
$\delta A=F\cdot dh$
$F=p\cdot S$
$\delta A=p\cdot \underbrace{ S\cdot dh }_{ dv }=p\cdot dv$
$$
A=\int_{V_{1}}^{V_{2}} \delta A \, =\int_{V_{1}}^{V_{2}} p \, dV  
$$
Если $A > 0 \implies$ газ расширился
